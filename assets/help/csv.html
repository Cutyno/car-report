General

The whole database gets exported with one file for each table. This means the data of one refueling is separated into multiple files. You can find the car by looking up the value of the fuel_tank column in the Id column of the fuel_tanks table and repeat this for the car column.

Columns

cars

Id: [int]
name: [string]
suspended_since: [date]
color: [int] Have a look at the android color documentation for a more detailed explanation how the value is calculated.

fuel_tanks

Id: [int]
name: [string]
car: [int] the Id of the car.

fuel_types

Id: [int]
name: [string]

fuel_types_fuel_tanks

Id: [int]
fuel_tank: [int] the Id of the fuel tank.
fuel_type: [int] the Id of the fuel type.

other_costs

Id: [int]
car: [int] the Id of the car.
mileage: [int] Total distance (odometer).
price: [float]
recurrence: "ONCE|DAY|MONTH|QUARTER|YEAR [int]" Example: DAY 5 means every five days.
title: [string]
date: [date]
note: [string]
end_date: [date] End recurrence on this date.

refuelings

Id: [int]
partial: "true|false"
date: [date]
note: [string]
fuel_tank: [int] the Id of the fuel tank.
volume: [float]
fuel_type: [int] the Id of the fuel type.
price: [float]
mileage: [int] Total distance (odometer).
