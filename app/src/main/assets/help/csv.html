General

The whole database gets exported with one file for each table. This means the data of one refueling is separated into multiple files. You can find the car by looking up the value of the car column in the Id column of the cars table.

Columns

cars

Id: [int]
name: [string]
suspended_since: [date]
color: [int] Have a look at the android color documentation for a more detailed explanation how the value is calculated.

fuel_types

Id: [int]
name: [string]
category: [string]

other_costs

Id: [int]
car: [int] the Id of the car.
mileage: [int] Total distance (odometer).
price: [float]
recurrence: "ONCE|DAY|MONTH|QUARTER|YEAR [int]" Example: DAY 5 means every five days.
title: [string]
date: [date]
note: [string]
end_date: [date] End recurrence on this date.

refuelings

Id: [int]
partial: "true|false"
date: [date]
note: [string]
volume: [float]
fuel_type: [int] the Id of the fuel type.
price: [float]
mileage: [int] Total distance (odometer).
car: [int] the Id of the car.

reminders

Id: [int]
title: [string]
after_time: "DAY|MONTH|YEAR [int]" Example: MONTH 3 means after three months.
after_distance: [int]
start_date: [date]
start_mileage: [int]
car: [int] the Id of the car.
notification_dismissed: "true|false"
snoozed_until: [date]
