Fuel Tanks

A car can have more than one tank, if it uses gas (LPG) in addition to ordinary fuel for example.

Fuel Types

If you have a hybrid car (e.g. LPG and ordinary fuel) you add a different fuel type for each tank.

You can also use fuel types if your car has only one tank. If your can can use Super 95 and Super E10 for example, you can add both types and see them distinguished in the fuel price report.
